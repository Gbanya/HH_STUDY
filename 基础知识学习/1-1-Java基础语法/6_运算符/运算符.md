运算符：特定操作的符号。
表达式：由运算符连起来的式子交表达式。

对于一个整数的表达式，除法用的是整除，结果仍然是整数，只看商，不看余数。

注意：  1.不同数据类型进行运算，结果为范围大的那类；
	2.对于字符串String（首字母大写，但不是关键字），加号代表连接符。
	3.任何数据类型和字符串进行连接是，结果都便成了字符串。
自增的区别，前++：先加后用；厚++：先用后加。
【范围】只有变量能使用自增和自减，常量无法改变，不能使用。

#逻辑运算符：&&、||、！。
&&与||具有短路效果，如果根据左边已经可以判断得到的结果，那么右边代码将不再执行，节省性能。
int a=10;
System.out.println(3>4 && ++a<100);//右边不再执行
System.out.println(a)//a=10

一元运算符：取反、自增、自减；
二元运算符：+、赋值；
三元运算符：需要三个数据才能进行操作的运算符。

格式： 数据类型 变量名称=判断条件？表达式A：表达式B；
int a =10;
int b =20;
int max= a>b?a:b//取大值
