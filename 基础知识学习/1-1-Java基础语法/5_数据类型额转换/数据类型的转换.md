*数据类型不一样时，会发生数据类型转换。
#自动类型转换（隐式）：
	1.特点：代码不需要处理，自动完成；
	2.规则：数据范围从小到大。(与字节数不一定相关，如float范围比long大) = 100;

如:long num1 = 100;
   double num2 = 2.5F;
   float 怒骂 = 2.5L；
*强制类型转换（显示）：
	范围小的类型 范围小的变量名 = （范围小的类型）原本范围大的数据。
注意：
	1.强制转换可能发生精度损失、数据溢出；
	2.byte、short、char可以发生数据运算，例如加法；
	3.计算式会先被表示为int类型，然后计算；
	4.boolean不能发生数据转换
如： byte num4 = 40;
     byte num5 = 50;
     int res = num4+num5;//byte+byte―>int+int = int
