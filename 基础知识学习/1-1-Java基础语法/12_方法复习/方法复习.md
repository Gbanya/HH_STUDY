方法：
1，参数：进入方法的数据；
2. 返回值：从方法出来的数据。

定义方法的完整格式：
修饰符 返回值类型 方法名称（参数类型 参数名称，。。。）{
	方法体；
	return 返回值；
}
其中，return有两个作用，第一个停止当前方法，第二个将后面的返回值调用处。
注意：返回值和方法名称前的“返回值类型”保持一致。

#方法的三种调用格式。
1.单独调用，方法名称（参数）；sum(a,b)
2.打印调用；System.out.println(sum(a,b)）)
3.赋值调用。int num=sum(a,b);System.out.println("变量的值："+num)

#有返回值和无返回值的方法：
1.有返回值的方法可以使用单独调用、打印调用或赋值调用；
2.对于无返回值的方法，只能使用单独调用，不能使用打印或赋值调用。

【注意】：
1.方法定义在类中；
2.顺序无关；
3.方法有返回值，一定要return；
4.对于void没有返回值的方法，可以写return，但后面不能接返回值；
5.方法中可以有多个返回值，但必须保证只有1个return被执行。
