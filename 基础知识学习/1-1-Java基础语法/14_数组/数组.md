数组定义：是一种容器，可以存放多个数据。

特点：
	1.数组是引用数据类型
	2.数组中的多个数据，类型必须统一
	3.数组的长度在程序运行期间不能改变大小

数组的初始化，在内存中创建一个数组，并且赋默认值。

两种常见的初始化方式。动态初始化和静态初始化。
1.动态初始化：指定长度
2.静态初始化：指定内容 eg，数据类型【】 名称= new 数据类型【】{元素1，元素2，。。。}


数据类型[] 数组名称=new 数据类型[]

左侧数据类型：数组中保存的数据，全部统一一种类型；
左侧中括号：代表是一个数组；
右侧数据类型：与左侧保持一致。

省略格式：
数据类型【】 数组名称 = {元素1，元素2，。。。}
静态初始化没有直接写出长度，但任然回自动推算得到。

【注意】
直接打印数组名称，得到的是数组对应的：内存地址哈希值。
访问数组元素的格式：数组名称[索引值]

Java内存划分为5个部分。（电脑内存条中）

1.栈：（Stack）：存放的都是方法中的局部变量；方法的运行一定在栈当中
	局部变量：方法的参数，或者方法{}内部的变量；
	作用域：一旦超出作用域，即可从栈内存中消失。
2.堆（Heap）：凡是new出来的内容都放在堆中，
	堆内存里面的东西都有一个地址值：16进制；
	堆内存里的数据，都有默认值。
3.方法区（Method Area）：内存。class相关信息，保护方法的信息。
4.本地方法栈：域操作系统相关。
5.寄存器：域CPU相关。

【数组常见问题】
1.数组必须进行new初始化才能使用，否则会显示空指针异常；
2.数组一旦创建，程序运行期间，长度不可改变；

【数组的对称位置交换】遍历用一个索引，交换位置通常两个索引。什么时候停止交换，（1）min==max（奇数）（2）min>max（偶数）
综合下来可以===》》》min<max时交换
for(int min=0,max=array.length-1;min<max;min++,max--){
	int temp=array[min];
	array[min]=array[max];
	array[max]=temp;
}
